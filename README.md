## GitHub Copilot Setup

1. Go to [GitHub's Docs](https://docs.github.com/en/copilot/how-tos/set-up/install-copilot-extension?tool=vimneovim)
2. Follow the instructions to install and configure GitHub Copilot for Neovim.
